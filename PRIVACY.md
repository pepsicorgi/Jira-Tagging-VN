# Privacy Policy for Jira Tagging VN

**Last Updated: March 2026**

### 1. Data Collection & Usage
Jira Tagging VN does **not** collect, store, or transmit any personally identifiable information (PII) to external servers. 

### 2. Local Storage
All user-created templates, categories, and settings are stored locally on the user's device using `chrome.storage.local`. This data never leaves your browser.

### 3. Permissions
The extension requires access to `https://jira.secext.samsung.net/` solely to:
* Detect the current ticket reporter for smart-tagging.
* Inject automated sub-task tables and comment templates.

### 4. Third Parties
No data is shared with third parties, advertisers, or external analytics engines.
